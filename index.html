RP2040-LCD-0.96 Implementation

This repository provides example code and a modularised approach for using the RP2040-LCD-0.96. Finding clear documentation for this specific module can be difficult, so this project documents a reliable workflow.

The Challenge: These modules often ship with CircuitPython, which can sometimes lock the virtual drive.
The Fix: I switched the processor to MicroPython using the specific Waveshare UF2 firmware to maintain compatibility with my other Pico projects.

Working Examples Included

I have verified and modularised several display scripts to work over the SPI interface:

  • pico-LCD-0.96.py - Core display driver.
  • WS_0-96_160x80_MIN.py - Minimal footprint setup.
  • graphicstest.py - Full visual test (requires sysfont.py).

Rather than relying on monolithic scripts, I have split the functions into a separate module, providing a clean base for any future RP2040 projects.
